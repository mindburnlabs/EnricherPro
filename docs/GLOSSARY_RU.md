# Russian Glossary (Глоссарий)

Ensures consistent terminology across the application.

## Core Domain Entities

| English Term      | Russian Term          | Context / Notes               |
| :---------------- | :-------------------- | :---------------------------- |
| **Enrichment**    | **Обогащение**        | Process of finding data       |
| **Agent**         | **Агент**             | AI agent executing the task   |
| **Research Plan** | **План исследования** |                               |
| **Conflict**      | **Конфликт**          | Data mismatch between sources |
| **Resolution**    | **Разрешение**        | Choosing the correct value    |
| **Source**        | **Источник**          | URL/Page where data was found |
| **Evidence**      | **Доказательство**    | Snippet confirming a fact     |

## Product & SKU

| English Term            | Russian Term                | Context / Notes                                                                      |
| :---------------------- | :-------------------------- | :----------------------------------------------------------------------------------- |
| **SKU**                 | **Артикул**                 | Sometimes "SKU" is acceptable in tech context, but "Артикул" is safer for broad use. |
| **Toner Cartridge**     | **Тонер-картридж**          |                                                                                      |
| **Drum Unit**           | **Фотобарабан (драм-юнит)** |                                                                                      |
| **Yield**               | **Ресурс**                  | e.g., "1500 pages"                                                                   |
| **Compatible Printers** | **Совместимые принтеры**    |                                                                                      |
| **Cross-Reference**     | **Кросс-референс**          |                                                                                      |

## Statuses & UI

| English Term     | Russian Term         | Context / Notes            |
| :--------------- | :------------------- | :------------------------- |
| **Needs Review** | **Требует проверки** | Item needs human attention |
| **Pending**      | **В очереди**        | Waiting to start           |
| **Running**      | **Выполняется**      | In progress                |
| **Completed**    | **Готово**           | Successfully finished      |
| **Failed**       | **Ошибка**           | Stopped due to error       |
| **Retrying**     | **Повтор**           | Automatic retry            |
| **Manual Queue** | **Ручная очередь**   |                            |
| **No Data**      | **Нет данных**       | Empty state                |

## Units & Formatting

| Type           | Format         | Example            |
| :------------- | :------------- | :----------------- |
| **Page Yield** | `N стр.`       | 15 000 стр.        |
| **Weight**     | `g` -> `г`     | 2400 г             |
| **Dimensions** | `mm` -> `мм`   | 350 × 350 × 200 мм |
| **Count**      | `pcs` -> `шт.` | 1 шт.              |

## Research "Tone of Voice"

- **Objective:** "Perplexity-like", transparent, active.
- **Examples:**
  - "Searching for..." -> "Ищем источники..."
  - "Reading content..." -> "Анализируем содержимое..."
  - "Verifying data..." -> "Проверяем данные..."
